# sysdep

نظام إدارة الديون — حسابات العملاء والمواد مع Firebase Firestore.

## التشغيل

```bash
npm install
npm start
```

ثم افتح `http://localhost:3000`

## Firebase

1. أنشئ مشروع Firebase وفعّل **Firestore Database**
2. انشر قواعد `firestore.rules` من Firebase Console
3. ضع إعدادات المشروع في `firebaseConfig` داخل `app.js`
